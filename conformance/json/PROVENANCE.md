# JSON Provenance

- Spec source: RFC 8259
- Canonical link: https://www.rfc-editor.org/rfc/rfc8259
- Snapshot basis: repo-owned audit cases created on 2026-03-15

JSON does not have a single official upstream conformance corpus comparable to
`toml-test` or `yaml-test-suite`, so this directory uses a repo-owned audit
slice derived directly from RFC 8259 grammar and normative examples.
